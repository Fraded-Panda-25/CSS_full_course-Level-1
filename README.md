# 🎨 CSS Full Course — Level 1

![CSS Course Banner](https://copilot.microsoft.com/th/id/BCO.0ef66772-77e7-4c12-b599-16a9885da11a.png)

Welcome to **CSS Full Course — Level 1**!  
This repository is designed for beginners who want to learn **Cascading Style Sheets (CSS)** step by step. It covers the fundamentals of styling web pages, from basic syntax to practical examples.

---

## 📚 Course Overview
By the end of Level 1, you will be able to:

- Understand **CSS syntax** and how it integrates with HTML  
- Apply **selectors, properties, and values** to style elements  
- Work with **colors, backgrounds, borders, and text formatting**  
- Explore **box model concepts** (margin, padding, border)  
- Practice **basic layouts** using display properties  

---

## 🗂 Repository Structure
| File/Folder | Description |
